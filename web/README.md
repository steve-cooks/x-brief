# X Brief Web

Next.js frontend for rendering `latest-briefing.json` generated by the Python pipeline.

## Run

```bash
cd web
npm install
npm run dev
```

Open `http://localhost:3000`.

## Data source

`src/app/api/briefing/route.ts` reads:
1. `${X_BRIEF_DATA_DIR}/latest-briefing.json` (if set)
2. fallback repo path `../data/latest-briefing.json`

Generate data first:

```bash
cd ..
python -m x_brief.pipeline configs/example.json --from-scans --hours 36
```
